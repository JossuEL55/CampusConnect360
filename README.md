\# CampusConnect 360



Ecosistema funcional de integración para una red de colegios.



\## Descripción



CampusConnect 360 es una solución distribuida orientada a la integración de sistemas académicos, financieros, de asistencia, notificaciones y analítica.



El ecosistema utilizará:



\- .NET

\- React

\- YARP API Gateway

\- RabbitMQ

\- PostgreSQL

\- Docker Compose

\- Serilog

\- Seq

\- JWT



\## Estructura inicial



\- `backend/Gateway`

\- `backend/Services`

\- `backend/SharedKernel`

\- `frontend`

\- `infrastructure`

\- `docs`

\- `scripts`



\## Estado



Proyecto en fase de configuración inicial de infraestructura y entorno compartido.
